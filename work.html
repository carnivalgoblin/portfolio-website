ProChat

A mobile chat app built with React Native and developed with Expo. Users can authenticate anonymously using Google Firebase authentication. Google Firestore Database is used to set up a real-time database. Messages are also stored locally on the user's mobile device via Async Storage, allowing users to access their messages offline. Users can send messages and share images and their locations.
